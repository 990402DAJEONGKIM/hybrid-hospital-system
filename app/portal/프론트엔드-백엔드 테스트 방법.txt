터미널에서 아래 두 명령을 유지하면서 Live Server로 프론트엔드를 열면 됩니다:


# 1. PostgreSQL (재시작 시 매번 실행)
sudo service postgresql start

# 2. 백엔드 서버 (backend 폴더에서)
cd /workspaces/msp_hospital_project/app/portal/backend
uvicorn main:app --reload --port 8000
--reload 옵션을 붙이면 코드 수정 시 서버가 자동으로 재시작됩니다. 이제 브라우저에서 회원가입과 로그인을 실제로 해볼 수 있습니다.




이전에 백그라운드로 실행한 uvicorn이 아직 남아 있는 경우

pkill -f "uvicorn main:app" && uvicorn main:app --reload --port 8000
또는 포트를 점유한 프로세스 PID를 확인하고 직접 종료할 수도 있습니다:


lsof -ti:8000 | xargs kill -9
이후 다시 uvicorn main:app --reload --port 8000 실행하면 됩니다.
