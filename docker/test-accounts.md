## 통합 직원 포털 — http://localhost:8002

## 비고

- 직원 계정은 `must_change_password = FALSE` → 바로 사용 가능
- 환자 계정은 `must_change_password = TRUE` → 첫 로그인 시 비밀번호 변경 화면으로 이동
- 계정 잠금: 로그인 5회 연속 실패 시 30분 잠금


## 테스트 실행
처음 실행하는 경우:
docker-compose up --build


이미 한 번 실행한 적 있는 경우 (DB 볼륨 존재):
docker-compose down -v   # 기존 DB 볼륨 삭제
docker-compose up --build
